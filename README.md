#### Number 类型
######  int ---------> 整形
###### float -------> 浮点型
###### Python  只有 float
###### 其他语言中又分单精度 (float) 和爽精度 ( double )

```Python
type (2/2)  // 输出 float 1.0
type (2//2)  // 输出 int 1
#双 //  只保留整数部分
```
